# 공군체력단련장 골프장 안내 사이트

이 폴더는 **공군체력단련장 골프장 안내 사이트** 전용 프로젝트입니다.

## 기준 문서

- 상위 폴더의 `공군체력단련장-기획서.md`
- 이 폴더의 `기획서.md`

## 작업 원칙

- 육군체력단련장 콘텐츠를 섞지 않습니다.
- 국방부, 공군, 공군복지시설예약체계와 무관한 비공식 정보 안내 사이트로 제작합니다.
- 공식 예약 기능을 제공하지 않고, 공식 예약체계로 안내하는 정보 사이트로 구성합니다.
- 애드센스 승인을 위해 시설별 상세 페이지는 충분한 본문 콘텐츠를 포함해야 합니다.
- 예약, 요금, 이용 자격, 운영시간은 공식 예약체계·이용요금·휴장일 안내 링크와 함께 표시합니다.

## 주요 폴더

- `facilities/`: 공군 시설별 상세 페이지
- `css/`: 공통 스타일
- `js/`: 공통 스크립트
- `images/`: 이미지 자산
- `legal/`: 개인정보처리방침, 이용약관

## 수정과 배포

이 폴더는 바로 배포할 수 있는 정적 사이트입니다.

- 스타일 원본: `css/style.css`
- 배포용 스타일: `css/style.min.css`
- 스크립트 원본: `js/main.js`
- 배포용 스크립트: `js/main.min.js`

원본을 수정한 뒤에는 배포용 파일을 다시 압축하고, 모든 HTML의 자산 URL에 붙은 `?v=` 값을 새 내용에 맞게 갱신합니다. 현재 HTML은 압축 파일을 불러오며 CSS, JavaScript, 이미지는 장기 캐시를 사용합니다.

배포 전에는 내부 링크, 페이지별 H1, 제목과 설명의 중복, JSON-LD 문법, 모바일 표의 `data-label`, 이미지 크기, 문서 가로 넘침을 확인합니다.

실제 도메인이 정해지면 각 페이지에 canonical URL과 `og:url`을 넣고 `sitemap.xml`, `robots.txt`의 주소도 같은 도메인으로 맞춥니다. 공개 가능한 운영자 문의 주소도 실제 주소로 교체해야 합니다.

Apache 배포는 `.htaccess`, 정적 호스팅은 `_headers`의 보안·압축·캐시 설정을 사용합니다.
