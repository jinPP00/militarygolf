예약 방법

  1. 로그인

    본인 이용자번호와 비밀번호로 공군복지시설예약체계에 로그인합니다.

  2. 예약 메뉴

    상단 예약/조회 메뉴의 예약신청을 누르거나 메인 화면의 예약신청 버튼을 이용합니다.

  3. 체력단련장 선택

    원하는 체력단련장과 날짜, 경기 구분(9홀 또는 18홀)을 선택합니다. 휴장일과 부대행사는 예약할 수 없습니다.

  4. 시간과 동반자 입력

    운동 희망 시간, 캐디 이용 여부, 동반자 정보를 입력합니다. 대표자 본인은 동반자에 입력하지 않습니다.

  5. 신청 완료 점검

    예약신청 완료 메시지를 보고 예약/조회 메뉴에서 신청 또는 확정 상태를 대조합니다.

예약확정 우선순위

  • 이용 대상별 배정비율 안에서 배정되며, 체력단련장별·공군과 타군 배정비율은 다를 수 있습니다.
  • 당월 이용실적이 적은 이용자, 전월 이용실적이 적은 이용자, 연간 이용실적이 적은 이용자 순으로 반영됩니다.
  • 동일 순위에서는 해당 체력단련장 당월 신청횟수가 많은 이용자, 회원정보 생년월일 기준 연장자 순서가 고려됩니다.
  • 장기간 신청하지 않았다는 이유만으로 예약순위에 우선 반영되지는 않습니다.
